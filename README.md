Hardware Portfolio
Selected hardware design projects from commercial engagements and personal work, authored by Gustavo Puerari — Embedded Hardware Designer based in Curitiba, Brazil.

Projects
01 — LoRaWAN Sports Scoring System
Status: In production · 3 devices · Long-range, battery-powered
A three-tier hardware system for live court-side score tracking on tennis and padel courts. Battery-powered controller with multi-month runtime, gateway hardware with two connectivity options (Wi-Fi and Wi-Fi + Ethernet), and on-screen display integration.
Stack: STM32L0 · SX1262 · ESP32-S3 · ESP32-C3 · W5500

Status: Delivered · ⭐⭐⭐⭐⭐ on Upwork · European client

Compact MIDI controller board converting MIDI 1.0 input to I²C output for integration with embedded host systems.
03 — M.2 Modular Keyboard Pedal Platform
Status: Active engagement — in progress · European client
Hardware platform implementing the M.2 form factor for musical keyboard pedal modules. Main carrier board with expansion modules including Wi-Fi (ESP32-P4) and FPGA-based modules.
04 — IoT Lock for Sports Courts
Status: Delivered
Integration board combining a pre-built electromagnetic lock, ESP32 module, and I²C display for access control to tennis and padel courts. Deployed in the same facility ecosystem as Project 01.
